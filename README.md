# Learning Platform

Microservices-based e-learning platform built with:
- Next.js (Frontend)
- Node.js / FastAPI (Backend)
- MongoDB, PostgreSQL, Redis
- Docker & Docker Compose
- AI Tutor & n8n automation

## Services
- User Service (Auth, JWT, Roles)
- Course Service
- Analytics Service
- AI Tutor Service

## Run project
```bash
docker-compose up -d
